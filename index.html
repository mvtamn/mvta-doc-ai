🖥 EA PC — PowerShell Network Sync

          
Save as C:\MVTA\MVTA_ID_Network_Sync.ps1 · Reads from EA's OneDrive sync folder → copies to network drive. Schedule via Task Scheduler every 15 minutes.

          

          

            
            
          


          
          

            Setup — Your Mac (do once, ~3 minutes)

            1. Open Terminal and run these three commands:

            mkdir -p /Users/tyrefant/MVTA/backup
            chmod +x /Users/tyrefant/MVTA/MVTA_ID_Sync.sh
            launchctl load ~/Library/LaunchAgents/com.mvta.idsync.plist
            2. From this point: every time you confirm a record, the .xlsx downloads to Downloads, and within 60 seconds the script copies it to:

            /Users/tyrefant/Library/CloudStorage/OneDrive-SharedLibraries-MVTA/Operations - Documents/_SOPs/_Drafts/Raw CSV/
            3. OneDrive syncs to SharePoint automatically within 1–2 minutes.


            Setup — EA's PC (do once)

            1. Edit MVTA_ID_Network_Sync.ps1 — set $OneDriveSource and $NetworkDest

            2. Open Task Scheduler → Create Basic Task → name: MVTA Document ID Sync

            3. Trigger: Daily, repeat every 15 minutes for 1 day

            4. Action: Start a program → powershell.exe

            5. Arguments: -ExecutionPolicy Bypass -File "C:\MVTA\MVTA_ID_Network_Sync.ps1"
